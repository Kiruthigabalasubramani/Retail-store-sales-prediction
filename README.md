# Retail Store Sales Prediction

This project focuses on predicting retail store sales using historical transaction data.
Machine learning models are used to analyze sales patterns and estimate future sales values.

## Dataset
The dataset contains retail transaction information such as customer details, store name,
product information, quantity, pricing, discounts, and final sales amount.
The dataset is used only for academic purposes.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Models Implemented
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Project Structure
- data/       : Dataset used for the project
- notebook/   : Jupyter notebook with implementation
- report/     : Academic project report reference
- requirements.txt : List of required Python libraries
